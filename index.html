Features & Technologies

Stable 200 CPS

High-precision winmm.dll multimedia timer implementation locking clicks perfectly up to 200 CPS with zero latency.

XML Theming

Full XML theming support. Create, edit, and share your own interface design configurations.

Mica & Acrylic

Native integration of Mica and Acrylic blur effects to perfectly match the Windows 11 design language.

Legacy Systems Support

Stable backward compatibility with older operating system builds without any drop in performance.
